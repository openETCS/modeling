\chapter{D3.6.1 Deliverable}
D3.6 deploys the SRS Subset 26 Functional Model.

The functional model provides a semi-formal model of the Subset 26 of the SRS. The deliverable covers the scope of the first iteration of the openETCS project (Kernel Functions). It is aligned with deliverable D3.5.1 (the Architecture).

The deliverable consists of the Scade Model, C-Code generated by the C-Code generator and a text document - also generated from the Scade Model.

You find the first iteration release on Github when you following this link: \url{https://github.com/openETCS/modeling/releases/tag/v0.1-D3.6.1}.

The First Iteration design document is stored under this link:
\url{https://github.com/openETCS/modeling/blob/master/openETCSDesignDocuments/firstIteration/WP3_InitialArchitecture_DesignDescription.pdf}
